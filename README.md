# 유틸리티 컬렉션

다양한 유용한 웹 기반 유틸리티 도구들의 모음입니다.

## 기능

1. XML 검증기
   - XML 문서의 유효성 검사
   - XML 스키마 검증
   - XML 포맷팅

2. 16진수-이미지 변환기
   - 16진수 문자열을 이미지로 변환
   - JPEG, PNG, GIF 등 다양한 이미지 형식 지원
   - 이미지 미리보기 및 다운로드

## 설치 및 실행

1. PHP 서버 실행
```bash
php -S localhost:8000
```

2. 브라우저에서 접속
```
http://localhost:8000/main.php
```

## 기술 스택

- PHP 7.4+
- HTML5
- CSS3
- JavaScript (ES6+)

## 디렉토리 구조

```
/
├── main.php           # 메인 페이지
├── css/              # 스타일시트 파일
├── js/               # 자바스크립트 파일
├── includes/         # PHP 유틸리티 함수
└── uploads/          # 업로드된 파일 저장
```
